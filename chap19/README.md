## 📝 学习建议

1. **理解双返回值**：掌握 value, ok := m["key"] 的用法
2. **掌握基本操作**：学会声明、初始化、赋值、取值
3. **理解引用类型**：了解 Map 是引用类型的特点
4. **实践应用**：学会用 Map 解决实际问题

## 🔑 核心知识点

- **双返回值**：区分"键不存在"和"值为零值"
- **声明语法**：map[KeyType]ValueType
- **引用类型**：Map 是引用类型，赋值是引用传递
- **逗号 ok 语法**：安全地检查键是否存在
- **预分配**：使用 make 预分配容量，提高性能
