## 📝 学习建议

1. **理解作用域**：理解不同级别的作用域
2. **掌握规则**：掌握作用域的规则
3. **避免遮蔽**：避免变量遮蔽问题
4. **实践应用**：在实际项目中应用作用域知识

## 🔑 核心知识点

- **作用域级别**：跨包全局 → 包内全局 → 函数内 → 块内
- **变量遮蔽**：内层变量会遮蔽外层同名变量
- **作用域规则**：变量在声明的作用域内有效
- **最佳实践**：合理使用作用域，避免命名冲突
