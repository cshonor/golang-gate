## 🔑 核心知识点

- **big.Int**：大整数，可以表示任意大小的整数
- **big.Float**：大浮点数，可以设置任意精度
- **big.Rat**：大有理数（分数）
- **SetString**：从字符串创建大数，支持不同进制
- **SetInt64**：从 int64 创建大数
- **new vs make**：new 用于值类型，make 用于引用类型
