# slice 数据结构

## 1. 三字段头（slice header）

在逻辑上，切片是对底层数组的 **视图**：

| 字段 | 含义 |
|------|------|
| **pointer** | 指向底层数组某起始元素 |
| **len** | 当前可见元素个数 |
| **cap** | 从起始位置到底层数组末尾可容纳的元素个数 |

## 2. 与数组的区别

- **数组** `[n]T`：值类型，长度是类型一部分。  
- **切片** `[]T`：描述符（头）+ **共享**底层数组。

## 3. 共享底层数组的后果

- 多个 slice 若共享同一数组，**一方 `s[i]=x` 可能影响另一方可见区间**。  
- `append` 若触发 **扩容**，可能 **换新数组**，与其它 slice **脱钩**（见 `slice扩容.md`）。
