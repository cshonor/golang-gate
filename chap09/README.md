## 📝 学习建议

1. **理解字符串**：理解字符串是不可变的
2. **理解字符**：理解字符、rune、byte 的关系
3. **掌握操作**：掌握字符串和字符的操作方法
4. **实践应用**：在实际项目中应用字符串知识

## 🔑 核心知识点

- **字符串类型**：string，不可变类型
- **字符类型**：rune（int32）和 byte（uint8）
- **Unicode**：Go 语言原生支持 Unicode
- **字符串遍历**：使用 range 遍历字符串得到 rune
- **字符串操作**：连接、切片、查找等操作
