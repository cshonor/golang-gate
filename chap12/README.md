## 📝 学习建议

1. **理解函数**：理解函数的基本概念
2. **掌握语法**：掌握函数的定义和调用

4. **实践应用**：在实际项目中应用函数知识

## 🔑 核心知识点

- **函数定义**：func 关键字，参数和返回值
- **可变参数**：使用 ... 表示可变参数
- **一等函数**：函数可以作为值、参数、返回值
- **闭包**：函数可以捕获外部变量
- **递归**：函数可以调用自身
