---

## 📝 学习建议

1. **理解结构体**：结构体是 Go 语言中自定义类型的基础
2. **掌握字段访问**：学会访问和修改结构体字段
3. **理解方法**：学会为结构体定义方法
4. **实践应用**：在实际项目中应用结构体

## 🔑 核心知识点

- **结构体定义**：type StructName struct { ... }
- **字段访问**：使用点号访问字段
- **结构体方法**：为结构体绑定方法
- **值类型**：结构体是值类型，赋值是值拷贝
- **组合**：通过嵌套实现组合
