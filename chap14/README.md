## 🔑 核心知识点

- **一等函数**：函数可以作为值、参数、返回值
- **函数类型**：函数也有类型，可以声明函数变量
- **闭包**：函数可以捕获外部变量
- **匿名函数**：没有名字的函数
- **捕获机制**：闭包会捕获外部变量的引用
