# 07 - 接口与多态实战

前置：[01-Go 接口（Interface）](./01-Go%20接口（Interface）.md)、[05-类型断言](./05-类型断言.md)。

---

## 1. 多态在 Go 里的形态

- **编译期**：变量静态类型是接口类型。  
- **运行期**：通过**动态类型 + 方法表**（`iface`）调用具体实现，即多态。  
- **无继承**：靠 **组合 + 接口** 表达「可替换行为」。

---

## 2. 最小示例：可替换的后端

```go
type Storage interface {
	Get(key string) (string, error)
}

type MemoryStore struct{}
func (MemoryStore) Get(key string) (string, error) { return "v", nil }

func Use(s Storage) { _, _ = s.Get("k") }
```

测试时可注入 `MemoryStore`，生产可换 `RedisStore`，只要满足 `Storage`。

---

## 3. 与标准库对齐

- `io.Reader` / `io.Writer`：只读或只写流，**小接口**大量组合。  
- `http.Handler`：`ServeHTTP` 一个方法，实现 Web 多态。
