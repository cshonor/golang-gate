## 1. 常见坑

| 坑 | 说明 |
|----|------|
| 对不可导出字段 Set | CanSet 常为 false |
| 忘记 Elem | 指针与值混淆 |
| 热路径滥用反射 | 慢且难优化 |
| 缓存 Type 不当 | 注意可比较性与用途 |

## 2. 实践

- 热路径避免反射；必须用时缓存 Type/方法查找。
- 初始化阶段反射，运行期查表。
