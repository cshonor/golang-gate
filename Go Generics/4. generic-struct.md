4. Go 泛型结构体
 
4.1 定义
 
泛型结构体：在结构体上定义类型参数，让同一个结构体可以存储、处理任意指定类型的数据，实现代码复用与类型安全。
 
4.2 基本语法
 
go  
type 结构体名[T any] struct {
    字段 T
}
 
 
4.3 最简单泛型结构体
 
go  
package main

import "fmt"

type Box[T any] struct {
	Content T
}

func main() {
	intBox := Box[int]{Content: 100}
	strBox := Box[string]{Content: "go"}
	fmt.Println(intBox, strBox)
}
 
 
4.4 泛型结构体 + 方法
 
go  
type Container[T any] struct {
	data T
}

func (c *Container[T]) Set(v T) {
	c.data = v
}

func (c Container[T]) Get() T {
	return c.data
}
 
 
4.5 多类型参数结构体
 
go  
type KeyValue[K comparable, V any] struct {
	Key   K
	Value V
}
 
 
4.6 常用示例：泛型栈
 
go  
type Stack[T any] struct {
	items []T
}

func (s *Stack[T]) Push(v T) {
	s.items = append(s.items, v)
}

func (s *Stack[T]) Pop() (T, bool) {
	if len(s.items) == 0 {
		var zero T
		return zero, false
	}
	item := s.items[len(s.items)-1]
	s.items = s.items[:len(s.items)-1]
	return item, true
}
