6. Go 泛型方法
 
6.1 什么是泛型方法
 
依附于泛型类型（泛型结构体/泛型接口）的方法，称为泛型方法。
泛型方法不能单独存在，必须绑定在带类型参数的类型上。
 
 
 
6.2 核心规则
 
1. 方法的接收者必须是泛型类型
2. 方法可以复用结构体的类型参数
3. 方法不能自己额外定义新的类型参数（只能用结构体已定义的）
 
 
 
6.3 基本语法
 
go  
// 先定义泛型结构体
type 类型名[T 约束] struct {
    字段 T
}

// 泛型方法：接收者使用泛型类型
func (接收者变量 类型名[T]) 方法名(参数) 返回值 {
    // 可以直接使用 T 类型
}
 
 
 
 
6.4 最简单示例：存取值
 
go  
package main

import "fmt"

// 1. 定义泛型结构体
type Box[T any] struct {
	value T
}

// 2. 泛型方法 Set（修改值）
func (b *Box[T]) Set(v T) {
	b.value = v
}

// 3. 泛型方法 Get（获取值）
func (b Box[T]) Get() T {
	return b.value
}

func main() {
	box := Box[int]{}
	box.Set(100)          // 调用泛型方法
	fmt.Println(box.Get()) // 输出：100
}
 
 
 
 
6.5 带指针接收者（必须掌握）
 
修改结构体内部数据时，必须用指针接收者  *Box[T] 
 
go  
// 正确：指针接收者，能修改原值
func (b *Box[T]) Set(v T) {
	b.value = v
}

// 错误：值接收者，修改的是副本，不生效
// func (b Box[T]) Set(v T) {
// 	b.value = v
// }
 
 
 
 
6.6 多类型参数的泛型方法
 
go  
// 多类型泛型结构体
type KV[K comparable, V any] struct {
	key   K
	value V
}

// 泛型方法：同时使用 K、V
func (kv KV[K, V]) Print() {
	fmt.Printf("key:%v, value:%v\n", kv.key, kv.value)
}
 
 
 
 
6.7 实用示例：泛型栈（Push / Pop）
 
go  
package main

import "fmt"

// 泛型栈
type Stack[T any] struct {
	items []T
}

// 入栈（泛型方法）
func (s *Stack[T]) Push(item T) {
	s.items = append(s.items, item)
}

// 出栈（泛型方法）
func (s *Stack[T]) Pop() (T, bool) {
	if len(s.items) == 0 {
		var zero T
		return zero, false
	}
	// 取最后一个元素
	item := s.items[len(s.items)-1]
	s.items = s.items[:len(s.items)-1]
	return item, true
}

func main() {
	s := Stack[int]{}
	s.Push(10)
	s.Push(20)
	fmt.Println(s.Pop()) // 20 true
}
 
 
 
 
6.8 泛型方法 VS 泛型函数
 
特性 泛型方法 泛型函数 
依赖 必须依附泛型类型 独立存在 
类型参数 复用结构体的 T 自己定义 [T any] 
调用方式 变量.方法() 直接调用函数() 
 
 
 
6.9 总结
 
1. 泛型方法 = 泛型类型 + 方法
2. 语法： func (b Box[T]) Method() 
3. 修改数据用指针接收者： *Box[T] 
4. 只能使用结构体已定义的类型参数
5. 常用于：容器操作（栈、队列、集合）
