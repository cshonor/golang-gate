3. Go 泛型函数
 
3.1 定义
 
泛型函数：使用类型参数，可以支持多种类型调用，编译时类型安全。
 
3.2 基本语法
 
go  
func 函数名[类型参数 约束](参数) 返回值 {
    函数体
}
 
 
3.3 任意类型泛型函数（any）
 
go  
package main

import "fmt"

func Print[T any](v T) {
	fmt.Println(v)
}

func main() {
	Print(10)
	Print("hello")
	Print(3.14)
}
 
 
3.4 带约束的泛型函数
 
go  
import "golang.org/x/exp/constraints"

func Max[T constraints.Ordered](a, b T) T {
	if a > b {
		return a
	}
	return b
}
 
 
3.5 自定义类型约束
 
go  
type Number interface {
	int | int64 | float64
}

func Add[T Number](a, b T) T {
	return a + b
}
 
 
3.6 多个类型参数
 
go  
func Pair[A, B any](a A, b B) (A, B) {
	return a, b
}
