## 一句话速记

- **数组 `[N]T`**：长度是**类型的一部分**，赋值/传参常是**整体拷贝**。  
- **切片 `[]T`**：拷贝/传参主要拷的是 **slice header**（指针+len+cap），底层数组可共享（见 `01-slice/`）。
