## 🔑 核心知识点

- **整数类型**：int8/16/32/64，uint8/16/32/64，int/uint
- **溢出回绕**：整数溢出会回绕到最小值或最大值
- **类型转换**：不同类型之间需要显式转换
- **big.Int**：处理超大整数
- **无符号陷阱**：无符号整数的减法可能产生大数
