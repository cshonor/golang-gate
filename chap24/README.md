- ✅ 接口的定义和声明
- ✅ 接口的实现

**学习目标**：掌握 Go 语言中接口的所有用法

---

## 📝 学习建议

1. **理解接口**：接口定义了一组方法的契约
2. **掌握实现**：学会实现接口
3. **理解多态**：理解接口的多态性
4. **实践应用**：在实际项目中应用接口

## 🔑 核心知识点

- **接口定义**：type InterfaceName interface { ... }
- **隐式实现**：Go 语言的接口是隐式实现的
- **多态性**：通过接口实现多态
- **空接口**：interface{} 可以表示任意类型
- **接口组合**：可以通过组合接口创建新接口
