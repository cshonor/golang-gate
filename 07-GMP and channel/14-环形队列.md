# 环形队列（Circular Queue）

基于**定长数组**实现的 **FIFO** 队列：用下标取模把首尾接成「环」，避免普通顺序队列的**假溢出**，空间利用率高。Go **有缓冲 channel** 的元素区就是典型应用（见 `hchan` 的 `buf`、`sendx`、`recvx`、`qcount`）。

## 1. 与 channel 的对应关系

| 概念 | 作用 |
|------|------|
| `buf` | 底层连续存储，长度对应 `dataqsiz` |
| `sendx` | 下一次**写入**的下标（队尾语义） |
| `recvx` | 下一次**读出**的下标（队头语义） |
| `qcount` | 当前元素个数，用于判空/判满 |
| `dataqsiz` | 容量；为 0 时表示无缓冲 channel，不走这段环形区 |

## 2. 空 / 满与下标推进

用 **计数** `qcount` 区分空与满（避免「队头队尾指针重合」歧义）：

- **空**：`qcount == 0`
- **满**：`qcount == dataqsiz`

下标更新（示意）：

```go
// 入队（写 buf[sendx] 后）
sendx = (sendx + 1) % dataqsiz
qcount++

// 出队（读 buf[recvx] 后）
recvx = (recvx + 1) % dataqsiz
qcount--
```

`% dataqsiz` 实现下标到尾后绕回 0，保证 **FIFO**。

## 3. 入队 / 出队（直觉步骤）

**入队（发送侧，缓冲未满时）**

1. 确认未满：`qcount < dataqsiz`
2. 写入 `buf[sendx]`，更新 `sendx`、`qcount`
3. channel 场景下可能唤醒 `recvq` 中的等待者（见 `chan.go`）

**出队（接收侧，缓冲非空时）**

1. 确认非空：`qcount > 0`
2. 读取 `buf[recvx]`，更新 `recvx`、`qcount`
3. 可能唤醒 `sendq` 中的等待者

## 4. 特点（实现与面试）

- **O(1)** 入队/出队（单次操作）。
- **连续内存**，对 CPU cache 相对友好。
- **固定容量**：需按峰值合理设 `make(chan T, n)` 的 `n`。

## 5. 注意点

1. **空/满**：不要单靠 `sendx == recvx` 推断（与计数法相比易错）；以 **`qcount`** 为准。
2. **并发**：裸环形缓冲区**不是**线程安全的；**channel** 用内置 `mutex` 保护整个 `hchan`。
3. **大元素 / 指针槽位**：出队后若需避免引用留住大对象，可将槽位置零（通用容器里常见；channel 实现有自己的类型与拷贝语义）。
4. **容量**：过大浪费内存，过小易阻塞、失去缓冲意义。

## 6. 与其他队列形态对比（概念）

| 特性 | 环形数组 | 普通顺序队列（整段移动） | 链表队列 |
|------|----------|----------------------------|----------|
| 假溢出 | 通过取模规避 | 易出现，空间浪费 | 动态扩展，一般无假溢出 |
| 单次入出 | O(1) | 出队可能 O(n)（挪动元素时） | 通常 O(1) |
| 内存布局 | 连续 | 连续 | 节点分散，cache 友好度视实现而定 |
| 典型场景 | channel 缓冲、定长 RingBuf | 小数据教学示例 | 动态长度需求 |

## 7. 参考实现（泛型 RingBuffer，需 Go 1.18+）

下面是与 `hchan` **思路相近**的练习用环形缓冲（**不等于** runtime 源码；channel 还有锁、`sudog`、关闭语义等）。

```go
package main

import (
	"errors"
	"sync"
)

type RingBuffer[T any] struct {
	mu       sync.Mutex
	buf      []T
	sendx    uint
	recvx    uint
	qcount   uint
	capacity uint
}

func NewRingBuffer[T any](capacity uint) *RingBuffer[T] {
	if capacity == 0 {
		panic("capacity must be greater than 0")
	}
	return &RingBuffer[T]{
		buf:      make([]T, capacity),
		capacity: capacity,
	}
}

func (r *RingBuffer[T]) Enqueue(item T) error {
	r.mu.Lock()
	defer r.mu.Unlock()

	if r.qcount == r.capacity {
		return errors.New("queue is full")
	}

	r.buf[r.sendx] = item
	r.sendx = (r.sendx + 1) % r.capacity
	r.qcount++
	return nil
}

func (r *RingBuffer[T]) Dequeue() (T, error) {
	r.mu.Lock()
	defer r.mu.Unlock()

	var zero T
	if r.qcount == 0 {
		return zero, errors.New("queue is empty")
	}

	item := r.buf[r.recvx]
	r.buf[r.recvx] = zero // 避免槽位长期引用大对象
	r.recvx = (r.recvx + 1) % r.capacity
	r.qcount--
	return item, nil
}

func (r *RingBuffer[T]) Len() uint {
	r.mu.Lock()
	defer r.mu.Unlock()
	return r.qcount
}

func (r *RingBuffer[T]) IsEmpty() bool {
	r.mu.Lock()
	defer r.mu.Unlock()
	return r.qcount == 0
}

func (r *RingBuffer[T]) IsFull() bool {
	r.mu.Lock()
	defer r.mu.Unlock()
	return r.qcount == r.capacity
}
```
