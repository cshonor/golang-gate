# 04 - 结构体偏移量与字段访问

## 1. Offsetof 用途

`unsafe.Offsetof(s.Field)` 可得到字段相对结构体起始地址的偏移。

## 2. 示例

```go
type User struct {
    ID   int64
    Age  int32
    Flag bool
}

var u User
off := unsafe.Offsetof(u.Age)
```

## 3. 应用场景

- 理解结构体对齐与内存布局
- 底层协议映射、序列化优化

## 4. 风险提醒

- 字段顺序一改，偏移就变
- 代码可读性显著下降，需加注释和测试
