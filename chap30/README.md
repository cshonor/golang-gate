## 🔑 核心知识点

- **goroutine**：轻量级并发执行单元
- **channel**：goroutine 之间通信的管道
- **流水线**：多个处理阶段通过通道串联
- **select**：多通道操作选择
- **并发模式**：扇出/扇入、工作池等常见模式
