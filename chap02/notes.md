# 第二章：美化的计算器

## 1. Import 导入包的规则

### 单个导入（不需要大括号）
```go
import "fmt"
```
- 如果只导入一个包，直接写 `import "包名"` 即可
- 不需要大括号

### 多个导入（需要大括号）
```go
import (
    "fmt"
    "math"
    "time"
)
```
- 如果要导入多个包，需要用大括号把包名括起来
- 每个包名占一行
- 最后一个包名后面可以有逗号，也可以没有（Go 会自动处理）

### 注意事项
- 一个文件只能有一个 `import` 语句块
- 不能混用两种方式（比如先写 `import "fmt"`，再写 `import ("math")`）

## 2. 基本数学运算

### 运算符
- `+` 加法
- `-` 减法
- `*` 乘法
- `/` 除法
- `%` 取余（模运算）

### 整数除法 vs 浮点数除法
```go
10 / 3        // 结果是 3（整数除法，截断小数部分）
10.0 / 3.0    // 结果是 3.333...（浮点数除法）
```

## 3. 火星计算示例

### 为什么说火星能让人"变年轻、变苗条"？

#### 体重计算（变苗条）
- **火星重力**约为地球的 **37.83%**
- 计算公式：`火星体重 = 地球体重 × 0.3783`
- 示例：164.0 磅 × 0.3783 = 62.04 磅

#### 年龄计算（变年轻）
- **火星一年**约 **687 个地球日**
- 计算公式：`火星年龄 = 地球年龄 × 365 ÷ 687`
- 示例：41 年 × 365 ÷ 687 ≈ 21.8 火星年

### 完整示例代码
```go
package main

import "fmt"

func main() {
    // 计算火星体重
    fmt.Println("你的重量在火星上:", 164.0*0.3783)
    
    // 计算火星年龄
    fmt.Println("你的年龄在火星上:", 41*365/687)
}
```

## 4. Println 打印函数

### 基本用法
```go
fmt.Println("文本内容")           // 打印文本
fmt.Println(123)                // 打印数字
fmt.Println("结果:", 10+5)      // 打印文本和计算结果
```

### 多个参数
```go
fmt.Println("a =", 10, "b =", 20)  // 会自动在参数之间加空格
// 输出：a = 10 b = 20
```

## 5. 总结

### Import 规则
- **单个导入**：`import "fmt"`（不用大括号）
- **多个导入**：`import ( "fmt" "math" )`（用大括号）

### 数学运算
- Go 支持基本的四则运算
- 注意整数除法和浮点数除法的区别

### 打印输出
- 使用 `fmt.Println()` 打印内容
- 可以打印文本、数字、表达式结果

### 实际应用
- 通过有趣的计算例子（如火星体重、年龄）学习基础语法
- 比单纯学理论更有趣、更容易理解
