## 🔑 核心知识点

- **竞态条件**：多个 goroutine 同时访问共享数据导致的不确定行为
- **互斥锁**：通过独占访问保护共享数据
- **应答通道**：通过通信来共享内存，避免直接共享
- **服务循环**：持续处理请求的并发服务模式
- **最佳实践**：选择适合的同步机制
