4. **实践应用**：在实际项目中应用组合

## 🔑 核心知识点

- **没有类**：Go 语言没有类的概念
- **组合机制**：通过结构体嵌套实现组合
- **代码复用**：通过组合实现代码复用
- **灵活性**：组合比继承更灵活
