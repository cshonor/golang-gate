# Go 指针类型 *T

## 1. 指针是什么

- 存储变量**内存地址**的类型
- 语法：*T 指向 T 类型的指针

## 2. 核心操作

- &x 取地址
- *p 解引用

```go
a := 10
p := &a
fmt.Println(*p) // 10
```

## 3. Go 指针限制（安全）

- 不支持指针运算（p++ 非法）
- 不支持任意类型转换
- 不会出现野指针滥用

## 4. 用途

- 避免大对象拷贝
- 修改函数入参
- 构建数据结构（链表、树）

## 5. 与 Java 对比

- Java 引用 = 安全版指针
- Go 指针 = 更底层、更可控
