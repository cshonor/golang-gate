# Go 基本类型总览

## 1. Go 类型体系两大分类

1. **基础类型（值类型）**
   - 数值类型（int, uint, float, complex）
   - 布尔类型 bool
   - 字符串 string
   - 指针类型 *T

2. **复合类型**
   - 数组 [n]T
   - 切片 []T
   - map map[K]V
   - 结构体 struct
   - 函数、通道、接口等

## 2. Go 类型核心特点

- **所有类型都是值类型**，赋值 = 复制
- 没有隐式类型转换，类型安全严格
- 每个类型都有**零值**，声明即初始化
- 类型本身就是安全边界，避免 Java/C++ 式隐式坑

## 3. 与 Java 的对应关系

- Go bool → Java boolean
- Go int/int64 → Java long/int
- Go uint → 无符号（Java 没有）
- Go string → Java String（但底层实现不同）
- Go *T → Java 引用（但更底层、更可控）
