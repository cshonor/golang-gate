# Go 数值类型

## 1. 整数类型

### 有符号整数

- int8   (-128 ~ 127)
- int16  (-32768 ~ 32767)
- int32  (-21亿左右)
- int64  (超大范围)
- int    (平台相关：32/64位，业务最常用)

### 无符号整数

- uint8, uint16, uint32, uint64
- uint (平台相关)
- byte = uint8（用于二进制、字节流）
- rune = int32（用于 Unicode 字符）

## 2. 浮点类型

- float32（单精度）
- float64（双精度，业务默认）

## 3. 复数类型

- complex64
- complex128

一般量化、科学计算才用，业务极少。

## 4. 典型声明

```go
var a int = 10
b := 20          // int
var c float64 = 3.14
```

## 5. 重点

- int 不是固定长度，跨平台要小心
- 做网络、存储、序列化时**必须用固定长度类型**（int32/int64）
