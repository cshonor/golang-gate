# Go Generics（泛型）

本目录整理 **Go 1.18+ 泛型**：类型参数、约束、泛型函数/类型/方法，以及与 `interface{}`、反射的取舍。示例中 **比较大小的约束**优先使用 **Go 1.21+** 标准库 **`cmp.Ordered`**；更早版本可用 `golang.org/x/exp/constraints` 或手写联合类型。

**前置**：熟悉接口与基本类型（可选 [02-interface-and-reflection](../02-interface-and-reflection/README.md)）。**错误返回习惯**见 [03-error_handling](../03-error_handling/README.md)。

**进阶笔记总入口**见仓库根目录 [README.md](../README.md)；**数据结构**见 [`../01-datastruct/README.md`](../01-datastruct/README.md)。

## 学习顺序建议

**主线**：`01 → 02 → 03 → 04 → 05 → 06 → 07 → 08 → 09 → 10 → 11 → 12`

1. **01–02**：为什么需要泛型、语法长什么样。  
2. **03–04**：泛型函数与泛型结构体。  
3. **05–06**：约束基础与进阶（`~`、组合、`comparable` 语义）。  
4. **07**：方法规则（接收者上的类型参数）。  
5. **08**：标准库泛型 API（日常最高频）。  
6. **09–10**：实现提要 + 避坑。  
7. **11–12**：小数据结构骨架 + 工程取舍（泛型 vs 接口）。

## 速记

- **类型参数**：`func F[T any](x T)`、`type List[T any] struct { ... }`  
- **约束**：`any`、`comparable`、`cmp.Ordered`（1.21+）、`~T`、自定义 `interface{ A | B }`  
- **何时不用泛型**：只在一处用到、或反射/插件化更合适时不必硬上
