# 08 - 泛型与标准库结合

**Go 1.21+** 起，`slices`、`maps`、`cmp` 等包提供大量**泛型 API**，是日常写业务时接触泛型最多的场景。前置：[03 - 泛型函数](./03-泛型函数.md)、[05 - 约束基础](./05-泛型类型约束基础.md)。

---

## 1. `slices`（切片工具）

```go
import "slices"

xs := []int{3, 1, 2}
slices.Sort(xs)
slices.Contains(xs, 2)
slices.Clone(xs)
```

- 排序、二分、裁剪、去重、折叠等均以**类型参数**实现，元素需满足相应约束（如 `cmp.Ordered` 或 `comparable`）。

---

## 2. `maps`（map 工具）

```go
import "maps"

m := map[string]int{"a": 1}
m2 := maps.Clone(m)
```

`maps` 中函数随版本增加（如键值遍历辅助等），以当前 Go **发行说明**为准；核心思路是 **`K comparable`** 约束下的复用。

---

## 3. `cmp` 包

`cmp.Compare`、`cmp.Less`、`cmp.Ordered` 与排序、优先队列、泛型 `Min`/`Max` 模板天然配套（见 [05](./05-泛型类型约束基础.md)）。

---

## 4. `errors` 与多错误

`errors.Join`（Go 1.20+）与泛型无直接语法关系，但在 **errgroup / 并发聚合** 场景常与「通用工具函数」一起出现；见 [03-error_handling](../03-error_handling/README.md)。

---

## 5. 使用建议

- 优先 **调标准库** 而非自造轮子；读函数签名可复习 `constraints` 与类型推断。
- 需要兼容旧版本时，对照 `golang.org/x/exp/slices` 等迁移路径。
