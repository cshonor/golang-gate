# 3. 泛型函数

泛型函数在**函数名后**声明类型参数，调用时通常由编译器**推断**类型实参。

## 1. 基本形式

```go
func 函数名[类型参数 约束](参数列表) 返回值 {
	// 函数体
}
```

## 2. `any`：任意类型

```go
package main

import "fmt"

func Print[T any](v T) {
	fmt.Println(v)
}

func main() {
	Print(10)
	Print("hello")
	Print(3.14)
}
```

## 3. 带约束：可比较大小（Go 1.21+ 用 `cmp`）

```go
import "cmp"

func Max[T cmp.Ordered](a, b T) T {
	if a > b {
		return a
	}
	return b
}
```

## 4. 自定义类型约束（联合类型）

```go
type Number interface {
	int | int64 | float64
}

func Add[T Number](a, b T) T {
	return a + b
}
```

## 5. 多个类型参数

```go
func Pair[A, B any](a A, b B) (A, B) {
	return a, b
}
```

---

## 复习速记

| 记什么 | 记一句 |
|--------|--------|
| 写在哪 | `func F[T any](...)` |
| 推断 | 多数情况调用处不用写 `[int]` |
