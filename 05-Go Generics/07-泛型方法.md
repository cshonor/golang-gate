# 07 - 泛型方法

**泛型方法**：接收者类型本身是**泛型类型**（如 `Box[T]`、`Stack[T]`）时，方法里可以继续使用类型参数 `T`（以及 `K,V` 等）。

## 1. 核心规则

1. 接收者必须是**带类型参数**的类型（如 `*Box[T]`、`Stack[T]`）。
2. 方法体里使用的类型参数，来自**类型定义**上的 `T`，与接收者一致。
3. Go 当前设计下，**不能**在普通非泛型类型上单独再给方法加一套全新的类型参数（与「泛型函数」不同）。

## 2. 基本形式

```go
type Box[T any] struct {
	value T
}

func (b *Box[T]) Set(v T) {
	b.value = v
}

func (b Box[T]) Get() T {
	return b.value
}
```

## 3. 完整可运行示例

```go
package main

import "fmt"

type Box[T any] struct {
	value T
}

func (b *Box[T]) Set(v T) {
	b.value = v
}

func (b Box[T]) Get() T {
	return b.value
}

func main() {
	box := Box[int]{}
	box.Set(100)
	fmt.Println(box.Get())
}
```

## 4. 指针接收者与值接收者

需要**修改**接收者内部字段时，一般用**指针接收者** `*Box[T]`，避免改到副本：

```go
func (b *Box[T]) Set(v T) { b.value = v } // 推荐
```

## 5. 多类型参数

```go
type KV[K comparable, V any] struct {
	key   K
	value V
}

func (kv KV[K, V]) Print() {
	fmt.Printf("key:%v, value:%v\n", kv.key, kv.value)
}
```

## 6. 泛型方法 vs 泛型函数

| | 泛型方法 | 泛型函数 |
|--|----------|----------|
| 依附 | 必须挂在泛型类型上 | 独立定义 |
| 类型参数 | 用类型上的 `T` 等 | 在函数名后写 `[T any]` |
| 调用 | `变量.方法()` | `函数(...)` |

## 7. 实用：栈的 Push / Pop

```go
type Stack[T any] struct {
	items []T
}

func (s *Stack[T]) Push(item T) {
	s.items = append(s.items, item)
}

func (s *Stack[T]) Pop() (T, bool) {
	if len(s.items) == 0 {
		var zero T
		return zero, false
	}
	i := len(s.items) - 1
	v := s.items[i]
	s.items = s.items[:i]
	return v, true
}
```
