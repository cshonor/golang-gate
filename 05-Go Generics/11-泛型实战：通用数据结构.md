# 11 - 泛型实战：通用数据结构

**核心一句话**：以前「`int` 栈、`string` 栈、某 `struct` 栈」要**重复写**；现在 **`Stack[T]` 写一次**，`Stack[int]`、`Stack[string]`……**全类型复用，还自带类型安全**（不用 `interface{}` + 断言）。

把 [04 - 泛型结构体](./04-泛型结构体.md)、[07 - 泛型方法](./07-泛型方法.md) 的语法落到**可复用容器**。以下均为**最小骨架**，便于粘贴后按项目规范补全（错误处理见 [03-error_handling](../03-error_handling/README.md)）。

---

## 新手先看：超级精简总结

- **泛型数据结构** = **结构不变**，**变的只是里面装的 `T`（和键 `K`）**。  
- 典型三个念头：**`Stack[T]`**、**`Node[T]` / `List[T]`**、**`Tree[K, V]`**（`K` 常带 **`cmp.Ordered`**）。  
- **并发**：容器**一般不加锁**；外面 **`sync.Mutex` 包一层**，职责更清晰。

---

## 1. 栈（`Stack[T]`，`slice` 后端）

```go
type Stack[T any] struct {
	items []T
}

func (s *Stack[T]) Push(v T) { s.items = append(s.items, v) }

func (s *Stack[T]) Pop() (T, bool) {
	var zero T
	if len(s.items) == 0 {
		return zero, false
	}
	i := len(s.items) - 1
	v := s.items[i]
	s.items = s.items[:i]
	return v, true
}
```

- **`T any`**：栈里元素类型由**调用方**定；`Stack[int]` 只装 `int`，**不会乱塞别的类型**。  
- **好处**：**类型安全**；逻辑集中在一处，不用为每种元素类型复制一整份栈。

**用法示例**：

```go
s := &Stack[int]{}
s.Push(1)
s.Push(2)
v, ok := s.Pop() // v == 2, ok == true
_, _ = v, ok
```

---

## 2. 单向链表（`Node[T]` / `List[T]`）

```go
type Node[T any] struct {
	Val  T
	Next *Node[T]
}

type List[T any] struct{ head *Node[T] }
```

- **每个节点**存一个 **`T`**；**`Next`** 指向下一个 **`Node[T]`**，整条链与具体是 `int` 还是 `struct` **无关**。  
- 插入/删除可写成 `*List[T]` 的方法；若需**可比较去重**，把 **`T any`** 改成 **`T comparable`**（见 [05](./05-泛型类型约束基础.md)）。

---

## 3. 二叉树（`Tree[K, V]`，`K` 可排序）

```go
import "cmp"

type Tree[K cmp.Ordered, V any] struct {
	root *node[K, V]
}

type node[K cmp.Ordered, V any] struct {
	key         K
	val         V
	left, right *node[K, V]
}
```

- **`K cmp.Ordered`**：键必须能 **`<` / `>`** 比大小（如 `int`、`string` 等），适合做**有序映射、查找树**上的比较分支。  
- **`V any`**：卫星数据**任意类型**。  
- 查找/插入按 **`cmp.Compare`** 或 **`<`** 分支实现即可（骨架略，按项目补全）。

---

## 4. 并发怎么用？（重点）

**泛型容器本身一般不加锁**；需要线程安全时，**外面包一层 `sync.Mutex`**，**结构更清晰**：泛型只管数据形状，并发策略由外层定（与 [09-concurrency](../09-concurrency/README.md) 配合）。

```go
import "sync"

type SafeStack[T any] struct {
	mu sync.Mutex
	s  Stack[T]
}

func (ss *SafeStack[T]) Push(v T) {
	ss.mu.Lock()
	defer ss.mu.Unlock()
	(&ss.s).Push(v) // Stack 的方法在 *Stack[T] 上
}

func (ss *SafeStack[T]) Pop() (T, bool) {
	ss.mu.Lock()
	defer ss.mu.Unlock()
	return (&ss.s).Pop()
}
```

> 实际项目里也可换 **`sync.RWMutex`**、**单 goroutine 拥有容器** 等模式；原则仍是：**别在通用容器里偷偷塞死锁语义**。
