# 11 - 泛型实战：通用数据结构

把 [04 - 泛型结构体](./04-泛型结构体.md)、[07 - 泛型方法](./07-泛型方法.md) 的语法落到**可复用容器**。以下均为**最小骨架**，便于粘贴后按项目规范补全（错误处理见 [03-error_handling](../03-error_handling/README.md)）。

---

## 1. 栈（slice 后端）

```go
type Stack[T any] struct {
	items []T
}

func (s *Stack[T]) Push(v T) { s.items = append(s.items, v) }

func (s *Stack[T]) Pop() (T, bool) {
	var zero T
	if len(s.items) == 0 {
		return zero, false
	}
	i := len(s.items) - 1
	v := s.items[i]
	s.items = s.items[:i]
	return v, true
}
```

---

## 2. 单向链表（节点泛型）

```go
type Node[T any] struct {
	Val  T
	Next *Node[T]
}

type List[T any] struct{ head *Node[T] }
```

插入/删除可写成 `*List[T]` 的方法；若需可比较去重，把 `T any` 改为 `T comparable`。

---

## 3. 二叉树（`cmp.Ordered` 键）

```go
import "cmp"

type Tree[K cmp.Ordered, V any] struct {
	root *node[K, V]
}

type node[K cmp.Ordered, V any] struct {
	key         K
	val         V
	left, right *node[K, V]
}
```

查找/插入按 `cmp.Compare` 或 `<` 分支实现即可。

---

## 4. 与并发

- **容器本身一般不加锁**；由调用方用 `sync.Mutex` 或 channel 包一层，避免在泛型类型里硬编码 goroutine 逻辑（与 [09-concurrency](../09-concurrency/README.md) 配合）。
