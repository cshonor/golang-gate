# 4. 泛型结构体

在 **type** 上声明类型参数，让同一套字段/方法适用于多种具体类型。

## 1. 基本语法

```go
type 结构体名[T 约束] struct {
	字段 T
}
```

## 2. 最简单例子

```go
package main

import "fmt"

type Box[T any] struct {
	Content T
}

func main() {
	intBox := Box[int]{Content: 100}
	strBox := Box[string]{Content: "go"}
	fmt.Println(intBox, strBox)
}
```

## 3. 泛型类型 + 方法

```go
type Container[T any] struct {
	data T
}

func (c *Container[T]) Set(v T) {
	c.data = v
}

func (c Container[T]) Get() T {
	return c.data
}
```

## 4. 多类型参数（如键值对）

```go
type KeyValue[K comparable, V any] struct {
	Key   K
	Value V
}
```

## 5. 常用：泛型栈

```go
type Stack[T any] struct {
	items []T
}

func (s *Stack[T]) Push(v T) {
	s.items = append(s.items, v)
}

func (s *Stack[T]) Pop() (T, bool) {
	if len(s.items) == 0 {
		var zero T
		return zero, false
	}
	i := len(s.items) - 1
	v := s.items[i]
	s.items = s.items[:i]
	return v, true
}
```

## 复习速记

| 记什么 | 记一句 |
|--------|--------|
| 声明 | `type Foo[T any] struct { ... }` |
| 实例化 | `Foo[int]{...}` |
