## 🔑 核心知识点

- **浮点数类型**：float32（单精度）和 float64（双精度）
- **精度问题**：浮点数有精度限制，不能精确表示所有小数
- **比较陷阱**：不能直接用 == 比较浮点数
- **误差容忍**：使用误差容忍度来比较浮点数
- **使用场景**：科学计算、图形处理等需要小数的场景
