## 建议学习顺序

1. **二选一入口**：先走 **01→05**（快/CAS/atomic/屏障），或先按 **06→13** 建立「flag → 自旋 → 阻塞 → futex → 饥饿」主线；两条线可交叉对照。  
2. 再读 **14**（`WaitGroup`）、按需读 **15**（`Gosched`）。  
3. 对照 **`sync.Mutex` 源码**（`src/sync/mutex.go`）与 **`runtime` 锁**（按需）。
