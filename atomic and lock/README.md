## 建议学习顺序

1. **二选一入口**：先走上方 **「闭环 1→5」**（快/CAS/atomic/屏障），或先按课表 **1 → 8** 建立「flag → 自旋 → 阻塞 → futex → 饥饿」主线；两条线可交叉对照。  
2. 再读 **`WaitGroup特性及原理.md`**（与锁并列的常用同步工具）。  
3. 对照 **`sync.Mutex` 源码**（`src/sync/mutex.go`）与 **`runtime` 锁**（按需）。
