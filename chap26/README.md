---

## 📝 学习建议

1. **理解指针**：理解指针的概念和作用
2. **掌握操作**：学会取址（&）和解引用（*）
3. **理解传递**：理解值传递和指针传递的区别
4. **实践应用**：在实际项目中应用指针

## 🔑 核心知识点

- **指针定义**：var p *Type
- **取址操作**：&variable 获取变量地址
- **解引用操作**：*pointer 获取指针指向的值
- **值传递 vs 指针传递**：理解两种传递方式的区别
- **nil 指针**：指针的零值是 nil
