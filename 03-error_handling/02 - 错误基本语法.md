# 02 - 错误基本语法

这是一份 **Go 语言错误处理的基础语法**笔记：标准返回、检查方式、创建错误，以及少量与后续章节衔接的要点。

---

## 1. 标准返回签名

```go
func Read(path string) ([]byte, error)
```

### 核心解读

- 这是 Go 里**最经典、最标准的函数返回值设计**：**把 `error` 放在最后一个返回值**。
- 含义：成功时返回业务数据（此处为 `[]byte`）和 `nil`；失败时返回类型的零值（如 `nil`）和具体错误。
- 设计上的常见说法：
  1. **显式错误处理**：调用者通常会立刻判断 `err`，比“隐式异常”更易跟踪控制流。
  2. **无 try/catch**：错误通过返回值传递，风格统一。
  3. **与标准库一致**：如 `os.ReadFile` 等普遍采用 `(T, error)`，便于形成团队习惯。

---

## 2. 标准检查写法

```go
data, err := Read("a.txt")
if err != nil {
    return err
}
```

### 核心解读

- 这是 Go 里**最常见的错误检查范式**。
- 流程：调用 → **先判断 `err != nil`** → 失败则返回（或记录日志、包装后再返回）→ 成功再使用 `data`。
- 习惯上注意：
  - 尽量避免用 `_` 丢弃错误（除非能证明安全且有意为之）。
  - 尽量在调用后**紧接着**检查，不要在未检查 `err` 时使用结果值。
  - 可在 `if err != nil` 分支里打日志、重试、转换为业务错误等，再 `return`。

## 3. 创建错误

基础方式主要有两种。

### 3.1 `errors.New("...")`

```go
return errors.New("file not found")
```

- 得到一个**仅含固定字符串**的简单 `error`。
- 适合：固定文案、无需动态拼接的场景。

### 3.2 `fmt.Errorf("...")`

```go
return fmt.Errorf("failed to read file %s: %v", path, err)
```

- 支持格式化，便于拼进路径、ID 等动态信息。
- **Go 1.13+**：可用 `%w` **包装**下层错误，保留错误链，供 `errors.Is` / `errors.As` 使用（详见 [04 - 错误包装与错误链](./04%20-%20错误包装与错误链.md)、[05 - errors.Is 与 errors.As 使用](./05-errors.Is%20与%20errors.As%20使用.md)）。

## 4. 与后续章节衔接的要点

### 4.1 错误包装与判断（简例）

```go
f, err := os.Open("a.txt")
if err != nil {
    return fmt.Errorf("open failed: %w", err)
}

// 上层
if errors.Is(err, os.ErrNotExist) {
    // 判断是否为「文件不存在」一类错误
}
```

更系统的包装、`Is`/`As` 用法见上文链到的 `04`、`05`。

### 4.2 自定义错误类型（简例）

复杂场景可用结构体实现 `error`，携带路径、操作名等上下文；展开写法见 [03 - 自定义错误类型](./03%20-%20自定义错误类型.md)。

```go
type FileError struct {
	Path string
	Op   string
	Err  error
}

func (e *FileError) Error() string {
	return fmt.Sprintf("%s %s: %v", e.Op, e.Path, e.Err)
}
```

### 4.3 常见误区（速查）

- 忽略错误：`data, _ := Read("a.txt")`（除非有充分理由）。
- 未检查 `err` 就使用 `data`。
- 无意义地层层包装同一句文案，日志重复难读。
- 用 `panic` 处理**可预期的业务错误**；`panic` 更适合不可恢复或编程错误（另见 [06 - panic 与 recover](./06-panic%20与%20recover%20捕获异常.md)）。

## 5. 完整示例（可运行）

```go
package main

import (
	"errors"
	"fmt"
	"os"
)

func Read(path string) ([]byte, error) {
	data, err := os.ReadFile(path)
	if err != nil {
		return nil, fmt.Errorf("os.ReadFile failed: %w", err)
	}
	return data, nil
}

func main() {
	data, err := Read("a.txt")
	if err != nil {
		if errors.Is(err, os.ErrNotExist) {
			fmt.Println("错误：文件不存在")
		} else {
			fmt.Printf("读取文件失败: %v\n", err)
		}
		return
	}
	fmt.Printf("读取成功，内容: %s\n", data)
}
```
