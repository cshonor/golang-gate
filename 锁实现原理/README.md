# 锁实现原理

本目录用于整理 **互斥锁 / 读写锁** 等同步原语的实现思路与对比（不限于 Go）。

## 建议收录主题

- **自旋锁（spin lock）**：忙等待、适用短临界区、避免上下文切换
- **互斥锁（mutex）**：阻塞等待、内核/用户态混合、公平性
- **读写锁（RWMutex）**：读多写少场景的优化与饥饿问题
- **Go `sync.Mutex` 演进**：正常模式 / 饥饿模式、自旋与阻塞的配合
- **与原子操作（atomic）的关系**：何时用锁、何时用原子

## 学习顺序建议

1. 临界区与竞态条件（为何需要锁）
2. 原子操作 + 内存序（happens-before）
3. 互斥锁的状态机与队列（等待、唤醒）
4. 读写锁与性能权衡

可将笔记、示意图、参考链接和示例代码放在本目录下的子文件中（如 `notes.md`、`examples.go`）。
