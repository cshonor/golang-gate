## 📝 学习建议

1. **理解布尔**：掌握布尔类型和比较运算
2. **掌握条件**：熟练使用 if 和 switch
3. **掌握循环**：熟练使用 for 和 range
4. **实践应用**：通过练习巩固知识

## 🔑 核心知识点

- **布尔类型**：true/false，比较运算符，逻辑运算符
- **if 语句**：条件判断，不需要小括号
- **switch 语句**：多分支选择，自动 break
- **for 循环**：三种形式，没有 while 关键字
- **range 循环**：遍历数组、切片、map、字符串
